# Yomio Dictionary Data

This repository contains the modified dictionary dataset used in the Yomio mobile application.

## Overview

The dataset is a processed and extended version of Japanese dictionary data derived from JMdict/EDICT and jmdict-simplified.

It is designed for efficient use in a mobile language-learning app. The dataset may be distributed in compressed gzip format. Decompress before use.

## Modifications

This repository contains a modified version of the source dictionary data.

Changes include:

- Conversion into a JSON format optimized for mobile applications
- Filtering and restructuring for app-specific usage
- Addition of Simplified Chinese (zh-CN) translations
- Addition of Traditional Chinese (zh-TW) translations
- Compression of generated data files for distribution

Changes have been made to the original content.
