# Yomio Dictionary Data

This repository contains the dictionary dataset used in the Yomio mobile application.

## Overview

The dataset is a processed and extended version of Japanese dictionary data, designed for efficient use in a mobile learning app.

## Sources

This dataset is derived from the following sources:

- **JMdict/EDICT Dictionary Project**  
  © Electronic Dictionary Research and Development Group (EDRDG)  
  https://www.edrdg.org/wiki/index.php/JMdict-EDICT_Dictionary_Project  

- **jmdict-simplified** by Dmitry Shpika  
  https://github.com/scriptin/jmdict-simplified  

## Modifications

The original data has been processed and extended with:

- Conversion into JSON format optimized for mobile applications  
- Filtering and restructuring for app-specific usage  
- Added **Simplified Chinese (zh-CN)** translations  
- Added **Traditional Chinese (zh-TW)** translations  
- This dataset constitutes a derivative work of the above sources
